# closing-checklist

An interactive closing checklist for end-of-shift use.

## Features

- Time-blocked tasks from 5:30 PM to 8:55 PM
- Tap any item to mark it complete
- Progress bar tracks completion across the full shift
- Add custom tasks to any time block during a shift
- Remove individual custom entries with the × button
- Reset button to uncheck everything for the next shift
- Clear custom entries button to remove any added tasks
- Custom entries are dashed-bordered to distinguish from defaults
- Fully responsive — works on desktop and mobile
